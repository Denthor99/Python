# MicroBlog: Un Proyecto de Blogging Ligero

MicroBlog es un proyecto de blog personalizado y ligero construido con el framework Django. Se trata de un proyecto muy sencillo, ya que con él he aprendido ha manejarme con este framework

## Características Principales

- **Interfaz Web Intuitiva**: Diseñada para ser fácil de usar, incluso para aquellos nuevos en el mundo del blogging. *Pendiente, cuenta con un diseño muy básico*
- **API RESTful Integrada**: Ofrece acceso programático a las funciones del blog, facilitando la integración con aplicaciones móviles y servicios externos.
- **Gestión de Usuarios**: Soporte para autenticación y autorización de usuarios.
- **Publicación de Entradas**: Los usuarios registrados pueden publicar entradas en el blog.
- **Comentarios**: Las entradas de blog permiten comentarios. *Pendiente*
- **Búsqueda y Filtrado**: Funcionalidades para buscar entradas de blog y filtrarlas. *Pendiente*

## Tecnologías Utilizadas

- **Django**: El framework de desarrollo web Python que sirve como base para el proyecto.
- **Django REST Framework**: Utilizado para construir la API RESTful del proyecto.
- **MySQL**: Base de datos utilizada para almacenar la información del proyecto.
